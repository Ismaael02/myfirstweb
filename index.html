  • Resumen

    Mi proyecto está formado por dos partes. Crear un home cinema con la raspberry Pi 4. Lo principal será comparar y comprar por amazon la Raspberry y los componentes necesarios para comenzar lo antes posible con el montaje e instalación de todo. Para el home cinema lo principal es que podamos ver online películas, series, documentales y escuchar música a través de nuestra raspberry. Una vez todo esto funcione correctamente instalaremos otro sistema operativo. Utilizaremos la misma tarjeta microSD donde está instalado el sistema operativo que normalmente utilizamos. El objetivo de este segundo sistema operativo es crear un servidor NAS para almacenar nuestros archivos. El servidor lo montaremos con Raspberry Pi Lite OS para asegurarnos que todo funciona correctamente y evitar problemas de compatibilidad. Mi conclusión es que podamos utilizar distintos sistema operativo por si en algún momento queremos aprovechar este potente mini ordenador para algo más que reproducir contenido multimedia.
